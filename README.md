
# BotiGitHubTest

Desafio do teste técnico para Dev Backend .Net


## Requisitos

- Docker
- Docker-compose
- SDK .Net Core 6.0


## Stack utilizada

- .Net Core 6.0



## Execução

Execute o arquivo [docker-compose.yml](https://github.com/cintyahx/GithubTopsters/blob/main/docker-compose.yml) com `docker-compose`

```bash
  docker-compose up
```
